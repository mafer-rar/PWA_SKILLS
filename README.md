Plataforma de Gamificación para Desarrollo de Habilidades Técnicas
==================================================================

> Aplicación Web Progresiva (PWA) diseñada para que los practicantes de Tata Consultancy Services (TCS) repasen y validen conocimientos técnicos básicos durante el periodo previo al inicio de su capacitación corporativa.

🎯 Descripción del Proyecto

**TCS Skills** es una plataforma educativa gamificada desarrollada como Aplicación Web Progresiva (PWA) que permite a los nuevos practicantes de TCS México reforzar sus conocimientos técnicos en tres áreas fundamentales:

1.  **Control de Versiones**: Git y GitHub
    
2.  **Lógica de Programación**: Algoritmos y estructuras de datos
    
3.  **Sintaxis Básica**: HTML, CSS y JavaScript
    

La aplicación transforma el repaso técnico tradicional en una experiencia interactiva mediante mecánicas de gamificación, permitiendo a los usuarios aprender a su propio ritmo, incluso sin conexión a internet.

### Problemática Resuelta

*   Reduce la pérdida de conocimientos técnicos durante el periodo de espera entre contratación e inicio de capacitación
    
*   Incrementa la motivación hacia el repaso técnico mediante elementos lúdicos
    
*   Proporciona una base de conocimientos homogénea para los nuevos ingresos
    
*   Disminuye el tiempo de mentoría requerido al inicio de las labores operativas
    

✨ Características Principales
-----------------------------

### 🎮 Gamificación

*   **Sistema de Puntos**: 100 puntos por cada respuesta correcta
    
*   **Barra de Progreso**: Visualización del avance por módulo en tiempo real
    
*   **Feedback Educativo Inmediato**: Explicación detallada tras cada respuesta
    
*   **Racha de Aciertos**: Registro de respuestas consecutivas correctas
    

### 📱 Experiencia de Usuario

*   **Diseño Responsivo**: Adaptable a desktop, tablet y móvil
    
*   **Interfaz Intuitiva**: Navegación clara y componentes visuales consistentes
    
*   **Modo Offline**: Funcionamiento completo sin conexión a internet
    
*   **Instalable**: Agregable a pantalla de inicio como aplicación nativa
    

### 🗄️ Gestión de Contenido

*   **Banco de Preguntas JSON**: Contenido editable sin modificar el código
    
*   **Categorías Temáticas**: Módulos independientes seleccionables
    
*   **Persistencia Local**: Progreso guardado en el dispositivo del usuario
    

### Instalación como PWA

1.  Abre la aplicación en Google Chrome (desktop o móvil)
    
2.  Busca el ícono de instalación en la barra de direcciones o el botón personalizado "Instalar App"
    
3.  Confirma la instalación
    
4.  La aplicación aparecerá en tu pantalla de inicio o menú de aplicaciones
    

### Requisitos del Navegador

*   ✅ Google Chrome 80+ (Recomendado)
    
*   ✅ Microsoft Edge 80+
    
*   ✅ Firefox con soporte PWA habilitado
    
*   ✅ Safari 13+ (soporte limitado para service workers)
    

> ⚠️ **Nota**: Para que las funcionalidades PWA (instalación y offline) funcionen correctamente, la aplicación debe servirse a través de **HTTPS** o **localhost**.
